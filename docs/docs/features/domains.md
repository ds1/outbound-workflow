# Domains

Manage your domain portfolio in Deep Outbound.

## Overview

The Domains page is your central hub for managing the domains you're selling. Each domain entry includes pricing, landing page URLs, and status tracking.

## Adding a Domain

1. Go to **Domains** in the sidebar
2. Click **Add Domain**
3. Fill in the details:

| Field | Description | Example |
|-------|-------------|---------|
| **Domain name** | The name without TLD | `weatherrobots` |
| **TLD** | Top-level domain | `com` |
| **Buy Now Price** | Your asking price | `$5,000` |
| **Floor Price** | Minimum acceptable | `$2,500` |
| **Landing Page URL** | Where buyers can purchase | `https://dan.spaceship.com/weatherrobots.com` |

4. Click **Save**

## Domain Status

Track where each domain is in the sales process:

| Status | Description |
|--------|-------------|
| **Available** | Listed for sale, accepting offers |
| **Reserved** | Temporarily held, not accepting offers |
| **Sold** | Transaction completed |
| **Expired** | No longer owned |

## Best Practices

### Pricing Strategy

- Set **Buy Now Price** at your ideal sale price
- Set **Floor Price** at your absolute minimum
- Consider market value and comparable sales

### Landing Pages

- Use professional landing pages (Spaceship, Afternic, Dan.com)
- Ensure the landing page shows current pricing
- Include multiple contact options

### Organization

- Keep statuses updated as situations change
- Archive sold domains for historical tracking
- Use consistent naming conventions
