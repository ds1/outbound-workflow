# Analytics

Track campaign performance and monitor costs.

## Overview

The Analytics dashboard provides insights into your outreach performance, lead funnel, and API usage costs.

## Campaign Metrics

### Performance Overview

Track key metrics across all campaigns:

| Metric | Description |
|--------|-------------|
| **Emails Sent** | Total emails delivered |
| **Open Rate** | Percentage of emails opened |
| **Click Rate** | Percentage with link clicks |
| **Reply Rate** | Percentage with responses |
| **Bounce Rate** | Percentage failed |

### Engagement Charts

Visual representations of:
- Opens over time
- Clicks over time
- Replies over time
- Comparison across campaigns

## Lead Funnel

Track leads through your sales process:

```
New → Contacted → Engaged → Qualified → Converted
```

### Conversion Rates

- **Contact Rate**: New → Contacted
- **Engagement Rate**: Contacted → Engaged
- **Qualification Rate**: Engaged → Qualified
- **Conversion Rate**: Qualified → Converted

## Cost Tracking

Monitor spending across all integrated services:

### Service Costs

| Service | Unit | Typical Cost |
|---------|------|--------------|
| **Claude API** | Tokens | ~$0.01 per generation |
| **ElevenLabs** | Characters | ~$0.30 per 1,000 chars |
| **Slybroadcast** | Voicemails | ~$0.04-0.05 each |
| **Resend** | Emails | Free up to 3,000/month |

### Cost Dashboard

View:
- Daily/weekly/monthly spending
- Breakdown by service
- Trend over time
- Budget vs. actual

### Cost Alerts

Set up notifications:

1. Go to **Settings** > **Notifications**
2. Enable cost threshold alerts
3. Set your monthly budget
4. Receive alerts when approaching limit

## Key Performance Indicators

### What to Track

**Email Campaigns:**
- Open rate benchmark: 20-30%
- Click rate benchmark: 2-5%
- Reply rate benchmark: 1-3%

**Voicemail Campaigns:**
- Delivery rate: 85-95%
- Callback rate: 1-3%

### Improving Performance

Low open rates:
- Improve subject lines
- Test different send times
- Clean your lead list

Low click rates:
- Clearer call-to-action
- More compelling offer
- Better personalization

Low reply rates:
- More targeted leads
- Stronger value proposition
- Follow-up timing
